EdgeLoop - Advanced Overlay Video Engine
==========================================

Version: v1.0.0-beta
Website: https://github.com/experiment-peepo/EdgeLoop

EdgeLoop is a high-performance, multi-monitor overlay video player designed for 
seamless, transparent playback.

INSTALLATION
------------
1. Extract the contents of this zip file to a folder of your choice.
2. Run 'EdgeLoop.exe' to start the application.

SYSTEM REQUIREMENTS
-------------------
- Windows 10/11 (x64)
- .NET 10 Desktop Runtime (Required)
- Hardware Acceleration support (Recommended)

DEPENDENCIES
------------
This package includes:
- yt-dlp.exe (for web video extraction)
- ffmpeg.exe (for video processing)

If you encounter issues with web videos, run the 'Dependencies/update_dependencies.bat' 
script to ensure you have the latest versions.
